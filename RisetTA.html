Rencana Pengembangan & Evaluasi

Strategi untuk membangun, menguji, dan memvalidasi keberhasilan sistem secara kuantitatif dan kualitatif.

Peta Jalan Proyek

Fase 1: Inisiasi

Generasi data sintetis & penyiapan lingkungan pengembangan.

Fase 2: Pengembangan

Pembangunan Backend API & Frontend UI secara paralel.

Fase 3: Pengumpulan Data

Pengumpulan data riil melalui kuesioner dan wawancara.

Fase 4: Evaluasi

Uji kuantitatif (NDCG) dan kualitatif (Usability Testing & CSAT).

Fase 5: Pengemasan

Kontainerisasi dengan Docker dan penyiapan paket CD.

Evaluasi Kuantitatif: NDCG

Normalized Discounted Cumulative Gain (NDCG) digunakan untuk mengukur kualitas peringkat. Metrik ini memberi nilai lebih tinggi jika item yang paling relevan (menurut ahli) berada di urutan teratas.
